Name: Image Classification project
Summary => what? and why?
=> The project:
I wanted to build a classifier using fastai library which can distinguish between daisy flowers and roses. 

Why?
Florida has a beautiful nature and different types of flowers so I wanted to do this project to build a big classifier for all the plants in florida with their information.



Features:
=> very simple interface, the user will click the button and choose an image to analyze whether it is daisy or rose
